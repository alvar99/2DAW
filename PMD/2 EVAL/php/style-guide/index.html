Guía de estilo para el diseño de T-Shop

El diseño de la web T-Shop esta realizado principalmente con la librería Bootstrap y con algunas modificaciones sobre esta libreria.

Para facilitar la comprensión de los estilos de la web se proporciona esta página donde se mostrará ejemplos de diseño de cada componente y una breve explicación de donde usarlo.

Colores

Los colores principales de la web són los oficiales de la NBA más el blanco y el negro.

Primario Secundario Negro Blanco
#1D4289 #C80F2E #303030 #FCFCFC
rgb(29,66,137) rgb(200,15,46) rgb(48,48,48) rgb(252,252,252)
NOTA: Tenemos que usar el color rojo con cuidado ya que puede dar la sensación de error o advertencia

Al hacer un uso principal de dos colores tan fuertes y contrastados como son el rojo y el azul, para destacar algo importante utilizaremos un color que junte a estos como es un morado pero para que llame la atención le daremos un toque más suave que estos dos

Botones

El diseño principal del botón es el de por defecto de bootstrap pero cambiando la modificación del color, la clase de bootstrap primary estará relacionada con nuestro color principal, la clase secondary con nuestro color rojo y la clase info con nuestro color morado.

Clases Ejemplo
class="btn btn-primary"
class="btn btn-secondary"
class="btn btn-info"
class="btn btn-dark"
class="btn btn-light"

Cartas de producto

La utilización de las cartas se hace para mostrar los diferentes productos de la tienda. Para el diseño en móbil la carta se vera con la imágen arriba y los datos debajo para tener una mejor visualización. En la versión web se vera la carta en formato apaisado con la imágen a la izquierda. El botón de acción siempre se situará en la parte inferior a la derecha.

El borde de la carta se le pondra los color principales para detacar la marca de la tienda.

Animaciones

Las animaciones son hechas de forma manual y vinculadas a una clase en concreto. Cada animación comienza dependiendo del evento que haga que salte, la gran mayoria comienzan al hacer over en el elemento pero hay otras animaciónes que sucenden al realizar alúna acción.

Para distinguir una animación el nombre de la clase en la que se asocia viene con el prefijo _a_ justo antes del nombre de la animación

Aquí recopilamos algúnas animaciones realizadas:

Clases Descripción Ejemplo
class="_a_buzz" El elemento hace un zumbido de lado a lado
class="_a_in-out" El elemento hace un zoom de más a menos
